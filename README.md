# Smart Farming Crop Yield Prediction

## Overview

This project predicts crop yield using agricultural and environmental factors such as rainfall, temperature, irrigation type, fertilizer type, and other farming parameters. The project also includes data visualization and machine learning techniques to analyze crop productivity.

## Features

* Load and analyze crop yield data.
* Handle missing values in the dataset.
* Perform exploratory data analysis (EDA).
* Visualize relationships between farming factors and crop yield.
* Build a Random Forest Regression model for crop yield prediction.
* Evaluate model performance using standard metrics.

## Dataset

The dataset contains information related to:

* Crop Type
* Rainfall
* Temperature
* Irrigation Type
* Fertilizer Type
* Soil Conditions
* Crop Yield (Target Variable)

## Visualizations

The project generates:

* Crop Yield Distribution Histogram
* Correlation Heatmap
* Crop Type vs Yield Box Plot
* Rainfall vs Yield Scatter Plot
* Temperature vs Yield Scatter Plot
* Irrigation Type vs Yield Box Plot
* Fertilizer Type vs Yield Box Plot

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Machine Learning Model

* Random Forest Regressor

## Evaluation Metrics

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* R² Score
